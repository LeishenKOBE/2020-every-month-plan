# 网络相关

1. 从输入 url 到页面呈现

```text
1. url输入回车
2. DNS解析(解析URL变成相应服务器的IP地址)
3. 浏览器网络向相应服务器发起TCP/IP连接请求（三次握手）
4. 建立TCP/IP连接
5. 浏览器网络发起HTTP请求
6. 等待响应过程
7. 下载HTML资源
8. 解析HTML
9. 遇到HTML里的资源，再次发起HTTP请求，下载资源（四次挥手）
10. 时间线
11. 呈现页面
```

2. 减少网络请求

```text
1. 雪碧图
2. base64编码图片
3. 合并脚本与样式表代码
4. 配置多个域名 CDN加速
5. 使用浏览器的缓存机制
```
